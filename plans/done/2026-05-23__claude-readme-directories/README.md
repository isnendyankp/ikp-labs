In progress.

## Summary

Add directory-level README files for Claude agents and skills so contributors can understand the purpose, structure, and usage of the `.claude/agents/` and `.claude/skills/` directories without opening every file first.

## Scope

### In Scope

- Add `.claude/agents/README.md`.
- Add `.claude/skills/README.md`.
- Document current agent families, triad roles, and skill directory structure.

### Out of Scope

- Rename existing agents or skills.
- Change agent permissions or settings.
- Add new agents, skills, hooks, or plugins.
