# Journals

**Time-based development logs** - Chronological records of development progress, daily checklists, and implementation summaries.

## What are Journals?

Journals are time-stamped entries documenting the development journey. They include:
- Daily/weekly progress logs
- Implementation notes from specific days
- Checklists and progress tracking
- Regression test reports
- Historical development decisions

## Structure

Journals are organized by year and month:

```
journals/
├── 2025-11/     # November 2025 entries
└── 2025-12/     # December 2025 entries
```
