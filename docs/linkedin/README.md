## Weekly Posting Workflow

1. **Complete weekly milestone** (feature, testing, etc.)
2. **Open template** ([linkedin-posting-template.md](template/linkedin-posting-template.md))
3. **Fill in sections** (Hook, Problem, Solution, Results, Tech Stack)
4. **Review character count** (target: 900-1,400 characters)
5. **Add to history** ([linkedin-posts-history.md](History/linkedin-posts-history.md))
6. **Post to LinkedIn** (optimal: Tuesday/Wednesday 9-10 AM)
7. **Track engagement** (update history with metrics)
